Method ao/detectOutliers

  DETECTOUTLIERS locates outliers in data.
 %%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
 
  DESCRIPTION: DETECTOUTLIERS locates outliers in ao objects.
 
 
  CALL:        out = obj.detectOutliers(pl)
               out = detectOutliers(objs, pl)
 
  INPUTS:      pl      - parameter list containing detection threshold
               obj(s)  - input ao object(s)
 
  OUTPUTS:     out - timeseries aos (one per input ao) corresponding to a
  flag for detected outliers (1 for outlier, 0 for normal data)
 
  Parameters Description
 
 %%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%

Parameter Description

Sets for this method …
Default

Default
no description
Key Default Value Options Description
detectOutliers
THRESHOLD 10 none Trigger threshold for detecting outliers. For Gaussian white noise with infrequent outliers, the units correspond to standard deviations.
CUSHION [] none Number of data points to include before outlier trigger start and after outlier trigger end. Effectively widens triggered area. Can either specify a single value or a 2-element array corresponding to pre- and post-trigger cushion.

Example
plist('THRESHOLD', [10], 'CUSHION', [[]])

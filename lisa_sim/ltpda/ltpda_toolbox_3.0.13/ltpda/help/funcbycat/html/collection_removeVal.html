Method collection/removeVal

  REMOVEVAL removes values from each AO in the ltpda_container.
 %%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
 
  DESCRIPTION: REMOVEVAL removes values from each AO in the ltpda_container.
 
  CALL:        out = removeVal(in, pl);
 
  Note: this is just a wrapper of ao/removeVal. Each AO in the ltpda_container is passed
  to removeVal with the input plist. 
  
  INPUTS:      in      -  input ltpda_container objects 
               pl      -  parameter list
 
  OUTPUTS:     out     -  output ltpda_container objects

Method collection/polyfit

  POLYFIT overloads polyfit() function of MATLAB for ltpda_container objects.
 %%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
 
  DESCRIPTION: POLYFIT overloads polyfit() function of MATLAB for ltpda_container objects.
 
  CALL:        out = polyfit(in, pl);
 
  Note: this is just a wrapper of ao/polyfit. Each AO in the ltpda_container is passed
  to ao/polyfit with the input plist. 
  
  INPUTS:      in      -  input ltpda_container objects 
               pl      -  parameter list
 
  OUTPUTS:     out     -  output ltpda_container objects

Method collection/iplot

  IPLOT calls ao/iplot on all inner ao objects.
 %%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
 
  DESCRIPTION: IPLOT calls ao/iplot on all inner ao objects.
 
  CALL:        obj = iplot(coll)
               obj = coll.iplot()
 
  Note: if the collection object does not contain AOs, then an error will be
  thrown.
